故事的开始

当你扫码到看到页面，发生了什么？

当你用微信扫描公众号二维码，或在浏览器输入网址按下回车的那一刻，一场跨越全球的数字旅程就已经启程。

📱 扫码识名

微信相机识别二维码图案，解析出其中的 URL 地址信息

第一步：二维码解析

🌐 域名寻路

DNS 系统将人类可读的域名翻译成计算机认识的 IP 地址

第二步：DNS 解析

🤝 握手建连

TCP 协议通过三次握手，确保客户端与服务器之间建立可靠连接

第三步：TCP 三次握手

🔒 加密传输

TLS 证书验证身份，非对称加密保护你的数据安全传输

第四步：HTTPS / TLS

🎉 页面呈现

服务器返回网页数据，浏览器解析渲染，你终于看到了这个页面

终点：恭喜完成旅程
